## CSS Hero Section Gradient

Hero section uses a dark gradient: **#1a1a2e** (top) to **#16213e** (bottom). Use these exact color values for consistency.

## Mobile Breakpoints

Responsive design uses mobile-first approach with breakpoints at:
- **900px**
- **768px**
- **600px**

## Framework and Language Policy

No JavaScript. Pure HTML5 and CSS3 only.
